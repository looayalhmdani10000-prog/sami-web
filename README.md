This is a simple static website for Al-Buraq Logistics (cargo & freight delivery). Files are separated:

Quick notes:
- The booking form in `booking.html` submits to `save_booking.php` (server-side PHP). Submitted bookings are stored in the `bookings` table in the local `alburaq` database.

Deploy:

- Netlify: Drag & drop the folder into Netlify or connect the repo for continuous deploy.
